# Smart-Traffic-Violation-Detection-System
Features:   - Vehicle Detection (YOLOv8/YOLOv11 via ultralytics)   - Vehicle Tracking (ByteTrack / BoT-SORT via supervision)   - Vehicle Counting   - Triple-Riding Detection on Motorcycles   - Speed Tracking (pixel-based estimation with calibration line)   - Speed Violation Detection (configurable speed limit)   - Violation Screenshot Capture
